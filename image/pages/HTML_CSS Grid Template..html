css html, what does this mean: grid-template-columns: 1fr 5fr;

grid-template-columns: 1fr 5fr; is a CSS property used to define the number and size of columns in a CSS grid layout.

The values 1fr and 5fr are the width of the columns in the grid. fr stands for "fraction" and refers to a unit of measurement that divides the available space in the grid into equal parts.

In this case, the first column will be one-fifth of the total available space and the second column will be five-fifths or the remaining four-fifths of the total available space.

So, if the grid container has a width of 1000px, the first column will be 200px wide and the second column will be 800px wide.
